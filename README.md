# cnctd_utils

Common utilities and helper functions for the cnctd ecosystem.

## Installation

```toml
[dependencies]
cnctd_utils = { git = "https://github.com/Connected-Dot/cnctd_crates.git" }
```

## Usage

```rust
use cnctd_utils::get_logo;

let logo = get_logo("cnctd");
println!("{}", logo);
```
